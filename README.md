# sylorei-plugins

Plugin marketplace for Claude Code.

## Install

```bash
claude plugin marketplace add SyloRei/claude-marketplace
claude plugin install claude-godmode@sylorei-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [claude-godmode](https://github.com/SyloRei/claude-godmode) | Production-grade engineering workflow: 8 agents, 8 skills, hooks, quality gates, and feature pipeline (`/prd` → `/plan-stories` → `/execute` → `/ship`) |
